# Syro

Syro 是一个面向 Obsidian 的间隔复习、笔记复习与知识增量工作流插件。

- 卡片复习
- 笔记复习
- 独立复习数据存储
- 基于 FSRS 的调度工作流
- 延期、重排程与统计工具
