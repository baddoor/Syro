# 快捷键索引

## 这是什么
- 这一页把文档主线里最重要的快捷键集中起来，尤其是闪卡复习会话中的键盘操作。

## 从哪里进入
- 闪卡复习会话。
- Cloze 创建命令。

## 适合什么场景
- 你希望减少鼠标操作。
- 你想把常见动作训练成肌肉记忆。

## 具体步骤
1. 在闪卡复习会话里优先记住：空格、数字键评分、撤销和与原文相关的常见操作。
2. 在内容编写里记住：`Create Cloze (Same Level)` 和 `Create Cloze (New Level)` 对应的快捷键（以当前实现为准）。
3. 真正训练时，一次只记一两组，不要试图一天背完全部。

## 相关设置 / 相关命令
- 相关页面： [复习会话](../flashcards/review-session.md)、[Cloze 工作流](../card-authoring/cloze-workflow.md)。

## 常见错误
- 先记快捷键，再去理解动作本身。
- 在不同工作流里混用同一套记忆方式。

## FAQ
- **所有命令都有默认快捷键吗**：不是。文档只强调当前实现里最重要、最值得记住的那一部分。
- **快捷键是不是比按钮更重要**：不是。先理解流程，再把高频动作迁移到键盘。
- **Cloze 快捷键为什么要最后学**：因为如果你还没理解同级和新级的语义，快捷键只会放大混乱。

## 排错与风险提示
- 错误的快捷键肌肉记忆会导致你在大量复习时反复误触，所以最好先小样本练习。
