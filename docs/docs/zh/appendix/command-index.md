# 命令索引

## 这是什么
- 这一页列出文档中经常提到的主要命令，并按工作流粗略分组，帮助你从命令面板快速定位。

## 从哪里进入
- 命令面板中搜索 `Syro`。

## 适合什么场景
- 你忘了命令面板里的精确名字。
- 你想把某个工作流映射回具体命令。

## 具体步骤
1. 笔记复习常看：`Open a note for review`、`Open Notes Review Queue in sidebar`。
2. 闪卡复习常看：`Review flashcards from all notes`、`Select a deck to cram`、`Review flashcards in this note`、`Cram flashcards in this note`。
3. 维护常看：`Rebuild Cache (Full Parse)`、`Global Sync Cards (Clean Ghost Cards)`。
4. 内容编写常看：`Create Cloze (Same Level)`、`Create Cloze (New Level)`。

## 相关设置 / 相关命令
- 相关页面： [所有入口](../getting-started/entry-points.md)、[快捷键索引](./shortcut-index.md)。

## 常见错误
- 把命令名和界面按钮名混为一谈。
- 只记得功能描述，却忘了命令面板里的实际英文名称。

## FAQ
- **为什么命令名是英文**：因为当前实现和翻译键在很多环境里仍以英文命令名更容易对照。
- **命令索引会包含所有内部命令吗**：不会，只包含文档主线里真正值得用户知道的命令。
- **我能把这些命令都设快捷键吗**：这取决于宿主环境和你自己的快捷键偏好。

## 排错与风险提示
- 命令名可能随版本微调；如果某个命令找不到，请先在命令面板里用关键词搜索。
