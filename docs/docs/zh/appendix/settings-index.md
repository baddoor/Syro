# 设置索引

## 这是什么
- 这一页按功能归纳设置项，帮助你在记不清页面结构时快速回到对应正文。

## 从哪里进入
- Flashcards、Notes、Algorithm、UI、Sync 相关设置。

## 适合什么场景
- 你记得某个开关的大概作用，但不记得它在哪个标签页。
- 你想把设置项映射回主教程章节。

## 具体步骤
1. 卡片解析、分隔符、Cloze 看 [卡片与笔记设置](../settings/flashcards-and-notes.md)。
2. 队列、忽略标签、Timeline 看 [卡片与笔记设置](../settings/flashcards-and-notes.md)。
3. FSRS、WMS 看 [算法与 WMS](../settings/algorithms-and-wms.md)。
4. 状态栏、调试、同步提示看 [界面与同步](../settings/ui-and-sync.md)。

## 相关设置 / 相关命令
- 相关页面： [设置总览](../settings/index.md)、[稳定设置与兼容字段](../settings/stable-vs-legacy.md)。

## 常见错误
- 只凭控件名字判断作用，不回到正文看完整语境。

## FAQ
- **为什么索引里不把所有兼容字段都展开**：兼容字段会把索引噪音拉得很高，所以主索引优先收纳用户真正会碰到的稳定项。
- **我怎么知道一个设置是不是兼容字段**：回到 [稳定设置与兼容字段](../settings/stable-vs-legacy.md)。

## 排错与风险提示
- 索引只能告诉你去哪一页，不能替你判断是否应该改。真正动手前还是要回正文确认。
