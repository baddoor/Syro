# Syro 使用教程

## 快速开始

推荐的基础配置：

- 数据位置：如果希望笔记更干净，优先使用独立数据文件
- 算法：FSRS
- 复习入口：命令面板、状态栏、侧边栏

启用插件后，先在命令面板中搜索 `Syro`，完成基础设置，再开始大规模导入或迁移。

## 核心工作流

### 复习笔记

你可以通过命令面板、状态栏或侧边栏进入笔记复习。复习过程中，Syro 可以通过浮动条等交互入口提供反馈操作。

### 复习卡片

你可以通过命令面板或卡片侧边栏开始卡片复习。Syro 同时支持普通复习和 CRAM 式集中复习。

### 添加复习内容

- 通过标签或组织方式让笔记参与复习
- 在支持的场景下将笔记或文件夹转换为复习分组
- 使用插件支持的卡片语法创建复习卡片

### 维护工具

Syro 提供以下维护类命令：

- 查看统计数据
- 延期笔记或卡片
- 重排已排程项目
- 查看条目复习元数据

## 迁移建议

如果你准备从其他复习插件迁移：

1. 先完整备份库。
2. 在同一个库中测试前，先禁用旧插件。
3. 明确复习数据是保留在笔记内，还是迁移到独立数据文件。
4. 最好先在副本库中验证迁移流程，再用于主库。

## 算法

Syro 支持多种调度策略，包括 FSRS。若你正在调参，不建议对同一批数据频繁切换算法。

- FSRS 参考：<https://github.com/open-spaced-repetition/ts-fsrs>

## 调试与恢复

部分维护命令面向调试或数据修复场景。执行重置、清理或大规模重排程前，请先保留备份。
